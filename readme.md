# Vibe

A music bot for discord

# Setup

`yarn install`

Create a `.env` file with the required fields.

run `ts-node ./src/utils/play-auth.ts` with youtube and spotify auth. ([play-dl auth instructions](https://github.com/play-dl/play-dl/tree/main/instructions))

`yarn deployCommands` to register the commands to discord (run this everytime commands are added or edited)

`yarn dev` to start the app in dev mode
